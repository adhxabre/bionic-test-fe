# Cypress

Cypress merupakan aplikasi automation testing yang biasanya digunakan untuk _end to end testing_, _integration testing_, dan _unit testing_

# Kubernetes

Kubernetes adalah suatu platform yang digunakan untuk memantau atau mengelola aplikasi yang terdapat pada kontainer

# ORM

ORM atau Object Relational Mapping adalah salah satu cara menggunakan tabel database dengan mengubahnya menjadi object yang lebih mudah untuk digunakan

# Class

Class berguna untuk membangun kerangka dasar pada bahasa pemprograman berorientasi objek

# Function

Function adalah struktur yang memperbolehkan kita membangun sebuah perintah kode yang kemudian menjadi sebuah fungsi yang dapat dijalankan

# DevOps

DevOps sendiri merupakan suatu proses yang dilakukan guna meminimalisir hambatan ketika tim developer membangun aplikasi

# Docker

Docker sendiri adalah sistem yang membantu kita membungkus suatu atau beberapa proyek dalam satu kontainer (container)
